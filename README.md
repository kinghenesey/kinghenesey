# 🚧 Featured Project

## 🌌 Nekova Programming Language

Nekova is a modern programming language designed with the following goals:

- Clean syntax
- High readability
- Developer-first design
- Modern language features
- Cross-platform development
- Powerful tooling
- Performance-focused architecture
- Scalable applications
- Formatter
- Debugger
- VS Code Extension
- CLI Tools
- Compiler
- Package Manager
- Standard Library
- NEKOVA Sandbox — isolated execution, resource limits, violation tracking

> Nekova is currently under active development, and new features are continuously being designed and implemented.

Future plans include:


- Language Server (LSP)
- Documentation Website
- Most importantly, Self-Hosting — NEKOVA lexer written in NEKOVA
- Building the NEKOVA Game Engine
